Desarrolladores

LibreBORME es software libre y su código fuente está accesible en la plataforma GitHub bajo la licencia Affero GPL v3.
La librería usada para extraer la información de los BORME se llama bormeparser y también es software libre. Está disponible en GitHub bajo la licencia GPL v3

Por ello puedes usar el código fuente de esta plataforma, estudiarlo, distribuirlo y mejorarlo (siempre respetando las licencias correspondientes).

API

LibreBORME cuenta con una API que hace posible integrar tus datos en aplicaciones y webs externas.
Puedes consultar la documentación en la plataforma ReadTheDocs y probarla hoy mismo.

Si tu aplicación va a realizar un gran número de peticiones a la API, contáctanos.

Contribuye

Una de las formas de apoyar LibreBORME es contribuir en su desarrollo.

En los issues de GitHub puedes encontrar las líneas de mejoras que hay abiertas, informar de errores en el software y aportar tus mejoras del código.
